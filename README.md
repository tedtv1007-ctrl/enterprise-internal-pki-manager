# Enterprise Internal PKI Manager

Automate internal certificate lifecycle management with Windows CA.

## Goal
The goal of this project is to provide a robust, automated solution for managing the lifecycle of internal certificates within an enterprise environment, leveraging Microsoft Active Directory Certificate Services (ADCS).

## Components
- **Center Control Portal**: The brain of the operation, managing logic and state.
- **CA Gateway**: The bridge to the Microsoft PKI infrastructure.
- **Collector**: The eyes and hands on the ground, discovering and deploying certificates.
